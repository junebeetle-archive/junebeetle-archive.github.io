macOS Mojave Patches

This page lists my contributions to dosdude1's Mojave patching project. They're already credited on this thread, though, so this page exists solely for my ego convenience.

I'm happy to say that I accomplished the following:
• Discovered how to boot Mojave on a Penryn-based system (refined by @jackluke as described here, analysis of another "solution" here just for fun)
• Enabled graphics acceleration on nVidia Tesla systems with a dumb hack, and later a less dumb one
• Fixed iSight camera on some MacBooks
• Fixed communication with Arduinos on old USB hardware
• Found how to re-enable acceleration on macOS 10.14.5 betas
• Improved @Minh_Ton's News patch by just editing a whitelist file rather than replacing the whole folder

These fixes have been incorporated into the patcher.
