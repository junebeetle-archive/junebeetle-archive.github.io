Uptime Script for BitBar

Download Script

I wrote this BitBar plugin a few years ago. It's a simple, reliable script to monitor your Mac's uptime. It also saves your best time so you can brag about how stable your machine is.

You need BitBar to use this. Install this script by copying it to your plugins folder. Enjoy!

When removing this script, you may want to delete the saved file here:

~/Library/Application Support/Best Uptime.txt
